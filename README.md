## **Chess Piece Labeler**

A Python desktop application for capturing images from a webcam, cropping chess pieces, and saving them into labeled folders for dataset creation.

## Features
- Live camera capture with OpenCV
- Display captured image in a Tkinter GUI
- Manual crop selection with mouse
- Save cropped images into separate folders for each chess piece
- Useful for creating datasets for computer vision and chess AI projects

## Tech Stack
- Python
- OpenCV
- Tkinter
- Pillow

## Use Case
This project helps build a labeled dataset of chess piece images for training image classification or object detection models.
